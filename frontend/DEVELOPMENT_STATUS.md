## 启动说明

### 1. 安装依赖
```bash
cd frontend
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问应用
打开浏览器访问 http://localhost:3000

### 4. 后端服务
确保后端Spring Boot应用运行在 http://localhost:8080
